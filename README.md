# Detailing Web

Car Detailing Booking System – React, TypeScript, and Tailwind CSS frontend for managing vehicle detailing appointments. Clients can browse services and book time slots, while admins access a dashboard to manage the system.

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- React Router

## Features

- Service catalog browsing
- Online booking interface
- Admin dashboard
- Responsive design

## Getting Started

### Prerequisites

- Node.js 18+
- [detailing-api](https://github.com/akaidalov/detailing-api) running locally

### 1. Install Dependencies

```bash
npm install
```

### 2. Start Development Server

```bash
npm run dev
```

The app will be available at http://localhost:5173

### 3. Build for Production

```bash
npm run build
```
